⚙️ API 연동 설정 가이드

아래 순서대로 설정하면 실시간 데이터를 사용할 수 있습니다.

Step 1 — 네이버 DataLab API
🟢 쇼핑 트렌드 실시간 조회
1. developers.naver.com 로그인
2. [Application] → [애플리케이션 등록]
3. 사용 API: DataLab(검색어트렌드) 체크
4. 발급된 Client ID / Secret을 .env 파일에 입력
Step 2 — Google Sheets 자동 저장
📊 선별 목록 자동 기록
1. sheets.new에서 새 스프레드시트 생성
2. [확장 프로그램] → [Apps Script] 열기
3. sheets-script.js 파일 내용을 붙여넣기
4. [배포] → [웹 앱으로 배포] → URL을 .env의 GOOGLE_SHEETS_WEBHOOK에 입력
Step 3 — 알리익스프레스 (중국 트렌드)
🔴 알리·테무 실시간 인기 상품
1. portals.aliexpress.com 어필리에이트 가입
2. 승인 후 App Key / App Secret 발급 (1~3일 소요)
3. .env의 ALI_APP_KEY, ALI_APP_SECRET에 입력
※ 테무는 공식 API가 없어 알리 데이터로 중국 직구 트렌드를 대표합니다.
Step 4 — 쿠팡 파트너스 (실시간 인기상품)
🛒 골드박스 · 카테고리 베스트
1. 쿠팡 파트너스 → 오픈 API → 키 발급/관리에서 ACCESS/SECRET 키 발급
2. Supabase Secrets에 COUPANG_ACCESS_KEY, COUPANG_SECRET_KEY 입력
3. api_cache 테이블 생성(0002 SQL) 후 함수 재배포
※ 쿠팡은 호출 제한이 엄격해 30분 캐싱으로 안전하게 가져옵니다.
Step 5 — 자동 시작 설정
⏰ 매일 아침 자동 실행
start-auto.bat 파일을 실행하면
Windows 작업 스케줄러에 매일 오전 7:30 자동 시작이 등록됩니다.

수동 시작: npm start → http://localhost:3000
